# Webpack The Right way


### Install

```bash
yarn install
```

### Development

```bash
yarn run start # create server on localhost:5000
```

```bash
yarn run build # build only
yarn run watch # build & watch
```


### Production

```bash
yarn install --prefer-offline
yarn run build:production
```

### Reports & analysis

```bash
yarn run report
```

```bash
yarn run cleanup
```
